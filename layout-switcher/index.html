The Flip plugin lets you seamlessly transition between two states even if there are sweeping changes to the structure of the DOM like re-parenting of elements which would normally cause them to jump to a new position/size.

All you need to do is get the initial state using Flip.getState(targets), make whatever changes you'd like to the DOM or styling, and then call Flip.from(state). GSAP will handle the rest!

"FLIP" is an animation technique that stands for "First", "Last", "Invert", "Play".

Now go "flip out" and have some fun with the new plugin.
